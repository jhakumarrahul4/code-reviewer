# AI Code Reviewer

A full-stack AI powered code review tool built with React and Node.js.

## Features
- Multi language support (JavaScript, TypeScript, Python, Java, C, C++)
- AI Code Review with quality score out of 100
- Automatic Bug Fixing
- Security Vulnerability Scanner
- GitHub file URL integration — load any public file directly
- Review History saved in localStorage
- Rate limiting and input validation
- Clean architecture with custom React hooks

## Tech Stack
- Frontend: React, Vite, Prism.js
- Backend: Node.js, Express.js
- AI: Meta Llama 3.1 via OpenRouter API
